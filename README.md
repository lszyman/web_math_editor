# Web math editor

## Instalacja

- Instalacja bower (`npm install -g bower`)
- Sciagniecie zaleznosci (`npm install`)
- Uruchomienie aplikacji (`npm start`)
- Uruchomienie testow (`npm test`)

## Strona glowna dostepna pod adresem (lokalnie):

http://localhost:8080
